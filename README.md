# Toastmasters Tools

https://cf25cf.github.io/toastmaster-namensschilder/

Web-App für Toastmasters Würzburg mit zwei Funktionen:

**Namensschilder** - Generator für Namensschilder in drei Kategorien (Vorstand, Mitglied, Gast). Erstellt eine druckfertige PDF mit 8 Namensschildern pro DIN-A4-Seite.

**Bewertungszettel** - Erstellt Bewertungszettel mit 3 anpassbaren Fragen. Eine Seite im Querformat ergibt 4 Zettel zum Ausschneiden.
